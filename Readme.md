# IWP Web Application

This project is a web-based application built using HTML, CSS, and Node.js. It includes multiple pages for academic management such as student information, faculty details, login, and registration. The backend is implemented with a Node.js server.

## 📁 Project Structure

```
IWP-main/
│
├── academics.html
├── faculty.html
├── forgetten.html
├── labs.html
├── login.html
├── overview.html
├── registerpage.html
├── students.html
├── style.css
│
├── server.js
├── package.json
├── .package-lock.json
│
├── students.json
└── users.json
```

## 🚀 Features

- User registration and login functionality
- Student and faculty information pages
- Academics and lab information pages
- Simple backend using Node.js
- JSON-based storage for users and students

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- Node.js
- JSON file storage

## 📦 Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd IWP-main
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the server**
   ```bash
   node server.js
   ```

4. Open your browser and visit:
   ```
   http://localhost:3000
   ```

## 🔑 Usage

- Navigate to the registration page to create an account.
- Login using the credentials.
- Access different pages for academics, students, faculty, and labs.

## 📂 Data Storage

The application stores data locally in JSON files:
- `users.json` → User credentials
- `students.json` → Student information
